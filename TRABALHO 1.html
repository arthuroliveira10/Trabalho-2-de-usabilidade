Affordance

Affordance refere-se às possibilidades de ação que um objeto ou ambiente oferece ao usuário[1]. No design de interfaces, affordances são pistas visuais que sugerem como um elemento deve ser usado.

Semiótica

A semiótica é a ciência que estuda os signos e os processos de significação[2]. Na interface digital, ela ajuda a compreender como os usuários interpretam símbolos e ícones.

Usabilidade

Usabilidade é a medida da facilidade de uso de uma interface[3]. Engloba eficiência, eficácia e satisfação do usuário ao realizar tarefas específicas.

CSS (Cascading Style Sheets)

CSS é uma linguagem de estilo utilizada para definir a apresentação de documentos HTML[4]. Permite controlar layout, cores, fontes e outros aspectos visuais.

Formas de Incluir CSS

  1. CSS Inline: Diretamente no elemento HTML usando o atributo style. Recomendado para estilos únicos e específicos.
  2. CSS Interno: Na seção head do documento usando a tag style. Útil para páginas únicas com estilos exclusivos.
  3. CSS Externo: Em um arquivo .css separado, linkado ao HTML. Recomendado para websites com múltiplas páginas, permitindo melhor manutenção e reutilização.
